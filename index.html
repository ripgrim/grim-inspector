Grim Inspector

A comprehensive debugging and inspection tool for web pages. This inspector provides real-time insights into your page's components, logs, errors, events, network activity, and performance metrics.

View Demo View on GitHub

Installation

Add this single line to your HTML file, right before the closing </body> tag:

<script src="https://cdn.jsdelivr.net/gh/ripgrim/grim-inspector@main/dist/grim-inspector.umd.js" type="text/javascript"></script>

Features

  • Components Tab: Lists all sections and JavaScript components
  • Logs Tab: Captures console logs for easy debugging
  • Errors Tab: Displays console errors with stack traces
  • Events Tab: Monitors DOM events and event listeners
  • Network Tab: Tracks API calls and fetch requests
  • Metrics Tab: Provides detailed performance metrics
